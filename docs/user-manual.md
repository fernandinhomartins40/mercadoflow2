# Manual do Usuario

## Dashboard
- Acompanhe vendas diarias, crescimento e top produtos.
- Alertas aparecem na pagina inicial e em "Alertas".

## Produtos
- Lista com receita, quantidade e preco medio.
- Filtragem por categoria (quando configurado).

## Market Basket
- Regras de associacao entre produtos.
- Use para campanhas e combinacoes de vendas.

## Configuracoes (Agente)
- Informe URL da API, token e ID do supermercado.
- Adicione as pastas onde os XMLs sao gerados.
- Use Start/Stop/Restart para controlar o servico.
- O Log Viewer permite filtrar erros e exportar CSV.
