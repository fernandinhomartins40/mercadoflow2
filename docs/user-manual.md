# Manual do Usuario

## Dashboard
- Acompanhe vendas diarias, crescimento e top produtos.
- Alertas aparecem na pagina inicial e em "Alertas".

## Produtos
- Lista com receita, quantidade e preco medio.
- Filtragem por categoria (quando configurado).

## Market Basket
- Regras de associacao entre produtos.
- Use para campanhas e combinacoes de vendas.

## Previsao de demanda
- Lista de previsoes por produto para os proximos dias (depende do job noturno).

## PDVs
- Cadastro de PDVs por supermercado (nome e serial opcional).

## Campanhas
- Cadastro de campanhas com periodo e descricao.

## Configuracoes (Agente)
- Informe URL da API e a `api_key`.
- A `api_key` e gerada no painel web em "Configuracoes" -> "Chave da API do coletor".
- O `market_id` pode ser preenchido automaticamente pela API (opcional).
- Adicione as pastas onde os XMLs sao gerados.
- Use Start/Stop/Restart para controlar o servico.
- O Log Viewer permite filtrar erros e exportar CSV.
